Introduction

In Jan 2020 I signed up for a code jam on the Python discord server. I was put on a team with 3 other people with the goal of creating an app in a week where the only restriction was that it had to be built using Kivy with the theme "ancient technology". We decided to do a morse code messaging app and we won first place in the competition beating out 18 teams. Here is a video showcasing some of the projects (our app is shown around 9:04) The app is also featured on Kivy's website.

One of the cooler feature it has was it's ability to listen to and live decode morse code messages, see demo here.
